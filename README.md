# CodeAlpha - Music Player 🎵

This is a functional music player created using HTML, CSS, and JavaScript as part of the CodeAlpha internship.

## 🚀 Features:
- Play / Pause / Next / Previous
- Display Song Title, Artist & Duration
- Progress Bar
- Volume Control
- Playlist & Autoplay

## 🛠️ Tech Stack:
- HTML
- CSS
- JavaScript

## 📁 Folder Structure:
index.html  
style.css  
script.js  
songs/ (mp3 files)
